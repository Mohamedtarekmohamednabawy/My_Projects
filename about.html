Difference between Get and post ?
Get retrieves data from a server and sends data via the URL, making it visible and suitable for {non-sensitive information}.
Post sends data to a server in the request body, keeping it hidden and suitable for {sensitive information}.
Get requests can be cached and bookmarked,while Post requests cannot.
